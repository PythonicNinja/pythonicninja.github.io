You would like to create type for following object with dynamic property name:

const gallery = {
  homepage: {
    key: 'gallery-of-users',
    activeSlide: 1,
  },
  feed: {
    key: 'gallery-of-posts',

  key: string;
  activeSlide: string;
}

  [slideKeyWhichIsDynamic:string]: IGalleryItem;
}

    ...
